Scratch Project

Scratch Game

This project is a game I made in Scratch for my APCS summer homework. Revit Architectural Model Project

This project is a laser cut house model I made in Revit in my Advanced Engineering design class. I spent a good amount of time learning how to use Revit, which is an architectural CAD program made by Autodesk, and modeling a floor plan I picked out in Revit. I spent the remainder of the time on this project converting the DXF elevation views into LightBurn, a laser cutting software, and laser cutting out the house pieces to assemble them. This project was very challenging because it took me a while to learn Revit and get comfortable with the controls. It was also challenging because there were some errors when converting the DXF files from Revit to LightBurn. But, I overcame this challenge by taking my time on learning Revit and by troubleshooting LightBurn. Overall, this project helped me learn how to use Revit and gain experience in making architectural models.

Congressional App Challenge 2022 Project

Organ Guessing Game

This project is an app created for the 2022 Congressional App Challenge. It's a computer game called "The Organ Guessing Game" and was created using Snap. It aims to teach elementary to middle school aged children about some of the main organs in the human body. The game consists of a matching game, where the player drags the organ name to the correct place on the anatomical model, where there's also a short definition of what that organ does. After the player successfully completes this task, they're prompted to take a short quiz to show what they've learned. The player gets to choose how many questions they answer. If their answer is correct, the program will display a message saying, "that's correct," and if their answer is incorrect, the program will display a message saying, "that's wrong," After the player has completed the quiz, the program will display their score and a button to retake the quiz will appear.
